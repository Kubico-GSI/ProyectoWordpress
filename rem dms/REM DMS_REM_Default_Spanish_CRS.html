Descripción El sistema deberá comportarse tal como se describe en el siguiente caso de uso cuando aparezca un plano en el proceso de venta al cliente
Precondición PD
Secuencia normal Paso Acción
1 El actor Cliente (ACT-0001) trae, envía o se le elabora un plano en la tienda
2 El actor Medidor (ACT-0003) se dirige al domicilio del cliente para revisar y realizar medidas más precisas sobre la zona que recibira la instalación de un producto. Posteriormente se le entrega al gerente
3 El actor Gerente (ACT-0002) entrega el plano al montador
4 El actor Montador (ACT-0004) se dirige al domicilio del cliente con el plano revisado y efectua el montaje
